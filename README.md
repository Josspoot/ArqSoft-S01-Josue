# Actividad 2

Este proyecto consiste en una aplicación web diseñada para la gestión y clasificación de videojuegos. Proporciona una interfaz sencilla donde el usuario puede registrar nuevos títulos y organizarlos dentro de un catálogo digital.

## Tecnologías Utilizadas

El desarrollo de la plataforma se basa en un stack tecnológico robusto que separa la lógica de negocio de la capa de presentación:

* **C#**: Lenguaje principal utilizado para la lógica del servidor y el manejo de datos.
* **.NET**: Framework de desarrollo empleado para estructurar la arquitectura de la aplicación web.
* **HTML**: Estructura fundamental de las vistas y componentes de la página.
* **CSS**: Definición de estilos, diseño visual y adaptabilidad de la interfaz de usuario.

## Características Principales

* **Registro de Títulos**: Formulario para ingresar información relevante sobre cada videojuego.
* **Clasificación**: Sistema para organizar los juegos según categorías o géneros.
* **Visualización de Catálogo**: Listado dinámico de los elementos almacenados en el sistema.

## Estructura del Proyecto

La arquitectura sigue los principios de desarrollo de software para mantener un código organizado y escalable:

1. **Controladores**: Encargados de gestionar las peticiones del usuario y coordinar las respuestas.
2. **Modelos**: Representan la estructura de los datos del videojuego.
3. **Vistas**: Archivos HTML/Razor que definen la experiencia visual del usuario final.

## Instalación y Ejecución

Para ejecutar este proyecto de manera local, es necesario contar con el SDK de .NET instalado.

1. Clonar el repositorio:
`git clone [https://github.com/Josspoot/ArqSoft-S01-Josue.git](https://github.com/Josspoot/ArqSoft-S01-Josue.git)`
2. Acceder al directorio del proyecto:
`cd ArqSoft-S01-Josue`
3. Restaurar las dependencias y ejecutar la aplicación:
`dotnet run`
